# Enterprise Company Intelligence Data Platform

## Overview
This project demonstrates an end-to-end Data Engineering pipeline built on GCP to ingest, process, and analyze enterprise company and financial data.

## Tech Stack
- Google Cloud Platform (GCS, BigQuery)
- PySpark
- SQL
- Python

## Architecture
- Batch ingestion from Cloud Storage
- Data processing using Spark
- Analytics-ready tables in BigQuery

## Use Case
Simulates real-world enterprise data pipelines used for company profiling, risk assessment, and analytics.
